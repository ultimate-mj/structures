```python
class DoublyLinkedList:
    def __init__(self, item):
        self.nodeCount = 0
        self.head = Node(None)  #추가
        self.tail = Node(None)  #추가
        self.head.prev = None
        self.head.next = self.tail  #추가
        self.tail.prev = self.head  #추가
        self.tail.next = None
```

### 리스트 순회

- 아래의 코드는 빈 리스트에 대해서도 유의함
```python
def traverse(self):
    result = []
    curr = self.head
    while curr.next.next:   #여기가 다름 #tail에도 dummy node가 있기 때문
        curr = curr.next
        result.append(curr.data)
    return result
```

#### 리스트 역순회

- 리스트 순회와 대칭적임

```python
def reverse(self):
    result = []
    curr = self.tail
    while curr.prev.prev:   #head에 dummy node가 있기 때문
        curr = curr.prev    #durlrk ekfma
        result.append(curr.data)
    return result
```

### 원소의 삽입

- 양방향으로 링크를 조절해줘야 하기 때문에 해줘야 하는 일이 많아 보이지만 실제 코드를 해보면 훨씬 유연하게 할 수 있음
- self.tail, self.head를 조정할 필요 없이 링크만 조정하면 되기 때문

```
--> prev ----------> next -->
             |
         --newNode--> 삽입

```

- 코드 순서
   1. newNode를 기준으로 양쪽 노드 얻어내기: next = prev.next
   2. newNode의 링크 조정: newNode.prev <- prev & newNode.next <- next
   3. 기존 링크 끊어주기: prev.next <- newNode & next.prev <- newNode
   4. nodeCount <- nodeCount + 1
 
 ```python
 def insertAfter(self, prev, newNode):
    next = prev.next
    newNode.prev = prev
    newNode.next = next
    prev.next = newNode
    next.prev = newNode
    self.nodeCount += 1
    return True
 ```

### 특정 원소 얻어내기

- 지난 번에 dummy가 있었던 일방향과 완전히 동일!!
```python
def getAt(self, pos):
    if pos <0 or pos > self.nodeCount:
        return None
    i = 0
    curr = self.head
    while i < pos:
        curr = curr.next
        i += 1
    return curr
```

- 이를 활용한 insertAt의 코드는 다음과 같다
```python
def insertAt(self, pos, newNode):
    if pos < 1 or pos > self.nodeCount + 1:
        return False
    prev = self.getAt(pos-1)
    return self.insertAfter(prev, newNode)
```

- 그런데, 리스트의 마지막에 원소를 삽입하면 처음부터 찾아가는 것이 비효율적임
- getAt을 다음과 같이 수정할 수 있다 (그러나 여전히 선형시간의 알고리즘임)
```python
def getAt(self, pos):
    if pos < 0 or pos > self.nodeCount:
        return None
    
    if pos > self.nodeCount//2:  #절반보다 뒤쪽이면 tail에서 찾아오도록
        i = 0
        curr = self.tail
        while i < self.nodeCount - pos + 1:
            curr = curr.prev
            i += 1
    
    else:
        i = 0
        curr = self.head
        while i < pos:
            curr = curr.next
            i += 1
        return curr
```

```python
def concat(self, L):
        self.tail.prev.next = L.head.next
        L.head.next.prev = self.tail.prev
        if L.tail.prev:
            self.tail.prev = L.tail.prev
        if self.head.next:
            L.head.next = self.head.next
        if L.tail.prev == None and self.head.next == None:
            return None
        self.nodeCount += L.nodeCount
```
